- **Programming Language**: Java
- **Web Technologies**: Java Servlet, JSP
- **Web Server**: Apache Tomcat
- **Frontend Technologies**: HTML, CSS
- **Build Tool**: Maven
- **Database**: MySQL
- **Version Control**: Git
- **Development Environment**: Eclipse
